Install Petra Wallet

To use MerkPoll as an election administrator, you need to install the Petra Wallet browser extension.

Step 1: Download Petra Wallet

Visit the official Petra Wallet website and download the extension for your browser.

Download Petra Wallet

Step 2: Create or Import Wallet

Follow the setup instructions to create a new wallet or import an existing one.

Step 3: Connect to MerkPoll

Return to MerkPoll and click "Connect Wallet" to authenticate as an administrator.
